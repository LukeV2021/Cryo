To push a file:
1. Navigate to directory using following command in CMD: cd C:\Users\meyer\OneDrive\Documents\Cryo
2. Add the all files using: git add 'filename' or add all files by using: git add .
3. (OPTIONAL) Use: git status to see which files are changed.
4. Commit the changes using: git commit -m "Commit Name" where commit name is a breif descriptoin of what is commited. 
5. Push the changes to the main branch using the command: git push -u origin main
